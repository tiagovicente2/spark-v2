Database API (Firestore-style)

Spark Database is a collection-based Document Store. It comes with zero-config client write access and real-time subscription mechanisms using WebSockets.

Initialization & Fetching:
// Reference a collection
const posts = spark.db.collection('posts');

// Fetch all documents in the collection
const allPosts = await posts.list();
console.log(allPosts);
Creating, Updating, Deleting:
// Create a new document (returns created document containing a generated id)
const newPost = await posts.create({
  title: 'My First Spark Site',
  content: 'Having an absolute blast vibe coding!',
  votes: 1
});

// Update a document by ID
await posts.update(newPost.id, { votes: 2 });

// Delete a document by ID
await posts.delete(newPost.id);
Real-Time Synchronization (Subscribing):
// Setup real-time listeners. Any writes/deletes will immediately trigger these
const unsubscribe = posts.subscribe({
  onCreate: (doc) => console.log('Document created:', doc),
  onUpdate: (doc) => console.log('Document updated:', doc),
  onDelete: (id) => console.log('Document deleted with ID:', id)
});

// Stop listening
unsubscribe();

AI Engine API (Gemini Client API)

Write client-side AI chat dialogues or generate visuals from a text prompt. The API keys are stored on the Spark server and proxied securely.

Chat Completions:
// Feed model with conversational thread
const answer = await spark.ai.chat([
  { role: 'user', content: 'Generate a short recipe for a developers lunch.' }
]);

console.log(answer);
Visual Prompting (Image Generation):
// Request image synthesis
const imageUrl = await spark.ai.generateImage('A neon lit server stack inside a glass vault');

// Render in DOM
const img = document.createElement('img');
img.src = imageUrl;
document.body.appendChild(img);

WebSockets API (Multiplayer Rooms)

Spark provides rooms where multiple client nodes can pass websocket events in real time. Perfect for collaborative designs, shared whiteboards, or multiplayer cursors.

Usage:
// Connect to custom room
const room = spark.room('game-lobby');

// Broadcast cursor coords to other clients in room
document.addEventListener('mousemove', (e) => {
  room.send('cursor-move', { x: e.clientX, y: e.clientY });
});

// Receive events from other players
room.on('cursor-move', (data, sender) => {
  console.log(\`User \${sender.name} moved cursor:\`, data);
  updateCursorPosition(sender.id, data.x, data.y);
});

// Cleanup / Leave Room
room.leave();

File Storage API

Upload files directly from client side forms and receive public image or resource CDN links.

Usage:
const fileInput = document.querySelector('input[type="file"]');

fileInput.addEventListener('change', async (e) => {
  const file = e.target.files[0];
  if (!file) return;

  // Upload and get URL back
  const publicUrl = await spark.storage.upload(file);
  console.log('File available at:', publicUrl);
});
